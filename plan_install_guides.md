# Implementation Plan: Install Guide MD Files

**Goal**: Replace `tool_install_package`'s real `pkexec pacman -S` execution with instructinal MD file generation. The tool tells the user it cannot install directly, creates an MD guide with the exact commands, and saves it to a configurable directory.

**Security closure**: This plan resolves C-1 (Critical) and H-3 (High) from `SECURITY_AUDIT.md` by removing the `pkexec` + `--noconfirm` code path entirely.

### Code details — `src/config.py`

def install_guides_dir() -> Path:
    """Output directory for generated install guide MD files."""
    cfg = load_config()
    path = cfg.get("install_guides", {}).get("directory", "install_guides")
    p = Path(path)
    if not p.is_absolute():
        p = PROJECT_DIR / p
    p.mkdir(parents=True, exist_ok=True)
    return p
```

Note: `mkdir(parents=True, exist_ok=True)` means the directory is auto-created on first call — no manual `install_guides/.gitkeep` needed.

**Summary**: Rewrite `_install_package` and `tool_install_package` in `src/tools/package_manager.py` to generate an MD file instead of calling `pkexec pacman -S`. Also remove `YAY_ANSWER_ALL` from the search function (M-5 from security audit).

### Code details — `src/tools/package_manager.py`

New `_install_package` logic:
```python
def _install_package(package_name: str) -> str:
    """Generate an install guide MD file (does NOT actually install anything)."""
    import datetime
    from src.config import install_guides_dir

    sanitized = re.sub(r'[^a-zA-Z0-9._-]', '_', package_name)
    timestamp = datetime.datetime.now().strftime("%Y%m%d_%H%M%S")
    filename = f"{sanitized}-{timestamp}.md"
    outdir = install_guides_dir()
    path = outdir / filename

    content = (
        f"# Install Guide: {package_name}\n\n"
        f"## pacman\n"
        f"```bash\n"
        f"sudo pacman -S {package_name}\n"
        f"```\n\n"
        f"## AUR (yay)\n"
        f"```bash\n"
        f"yay -S {package_name}\n"
        f"```\n\n"
        f"*Generated by GnomePilot — always verify packages before installing.*\n"
    )
    path.write_text(content)
    return str(path)
```

Updated `tool_install_package` docstring:
```python
@tool()
def tool_install_package(package_name: str) -> str:
    """Generate an install guide with exact commands for a software package.

    Does NOT install anything. Creates an MD file with pacman/yay commands
    that the user can run manually. Use after confirming the package name
    with tool_search_packages.

    Args:
        package_name: Name of the package to write instructions for.
    """
    return _install_package(package_name)
```

Remove `env={**subprocess.os.environ, "YAY_ANSWER_ALL": "1"}` from `_search_packages` — replace with just `env=dict(os.environ)` or drop the env override entirely.

# test_package_manager.py — new file
from src.tools.package_manager import tool_install_package

def test_install_guide_creates_md_file():
    result = tool_install_package("htop")
    assert result.endswith(".md")
    path = Path(result)
    assert path.exists()
    content = path.read_text()
    assert "sudo pacman -S htop" in content
    assert "yay -S htop" in content
    assert "not install" not in content  # actually let's not check negatives
    path.unlink()  # cleanup

def test_install_package_no_actually_install():
    # Verify no subprocess calls are made
    import subprocess
    original = subprocess.run
    calls = []
    def fake_run(*args, **kwargs):
        calls.append(args)
        return subprocess.CompletedProcess(args, 0, b"", b"")
    subprocess.run = fake_run
    try:
        tool_install_package("htop")
        assert len(calls) == 0, f"Expected 0 subprocess calls, got {len(calls)}"
    finally:
        subprocess.run = original

def test_install_guide_sanitizes_filename():
    result = tool_install_package("firefox-esr/stable")
    assert "../" not in result  # no path traversal
    assert "firefox-esr_stable" in Path(result).stem
```
